## 概要

若年層の読書離れという社会課題に対し、
本物の文章との出会いを軸に、読書体験を再設計する読書サービスを想定。

読んだ本や、心に残った言葉との出会いを重ねることで、
読書をその場限りの体験ではなく、
自分の内側に積み重なっていく時間へと変えていく。

振り返るたびに、新しい気づきや視点が生まれ、
読書が自分を育てていることを実感できる体験を通じて、
自然と読書が続いていく状態を目指す。

## ターゲット

10 代から 30 代の男女

## キャッチコピー

言葉と出会い、自分に出会う
振り返りと共有が、自分の変化に気づかせる
